About

Spark is a Firebase-hosted Flask app running on a Docker served by Google Cloud. It employs Cloud Firestore and Realtime Database to store information pertaining to users. Spark was created because we, RGBHack (a team of high school students), noticed the need for a platform that could bring increased productivity to teams during the quaratine due to COVID-19. This project is open source, so you can learn more by visiting our GitHub Repository.
